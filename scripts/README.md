# 🛍️ Myntra Clone — Modern E-Commerce Front-End Experience

A responsive and feature-rich **Myntra-inspired shopping platform** built using **Vanilla JavaScript, HTML5, and CSS3**. This project recreates the real-world flow of modern e-commerce applications including product discovery, filtering, wishlist management, shopping cart operations, and dynamic UI rendering.

Designed with a **modular architecture and client-side state management**, the application focuses on delivering smooth interactions while maintaining clean and scalable front-end structure.

## ✨ Key Features

### 🛒 Smart Shopping Experience

* Browse products through structured category sections
* Dynamic product rendering using reusable DOM generation logic
* Smooth navigation across shopping workflows

### ❤️ Wishlist & Bag Management

* Add or remove products instantly
* Persistent data storage using browser `localStorage`
* Cart and wishlist remain synchronized across pages

### 🔍 Real-Time Search & Filtering

* Instant search experience with live text matching
* Category-based filtering through URL query parameters
* Dynamic content updates without page refresh

### 💰 Dynamic Pricing Engine

* Automatic total price calculation
* Discount percentage evaluation
* Convenience fee integration
* Final payable amount generated in real time

### ⚡ Interactive UI Components

* Dynamic modal generation
* Responsive layouts for multiple screen sizes
* Smooth state transitions and DOM updates
